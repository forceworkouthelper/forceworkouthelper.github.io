Use this setting to select the language that the app must use.

We currently support the following:

- Afrikaans
- Arabic
- Chinese (Mandarin)
- Bengali
- English
- French
- German
- Hindi
- Indonesian
- Italian
- Japanese
- Korean
- Portuguese
- Russian
- Spanish
- Swahili
- Turkish
- Urdu
